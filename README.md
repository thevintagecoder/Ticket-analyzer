# Ticket Analyzer

Ticket Analyzer is a minimal full-stack application for submitting support
tickets, analyzing their sentiment, saving them in PostgreSQL, and displaying
ticket history.

## Planned architecture

- Frontend: React and Vite
- Backend: FastAPI
- AI model: Hugging Face sentiment-classification model
- Database: PostgreSQL
- Local orchestration: Docker Compose
